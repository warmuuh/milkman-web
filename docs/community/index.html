Join the Milkman community

Milkman is an open source project that anyone in the community can use, improve, and enjoy. We'd love you to join us! Here's a few ways to find out what's happening and get involved.

Learn and Connect

Using or want to use Milkman? Find out more here:

  • User Discussion Board: Discussion and help from your fellow users
  • Twitter: Follow us on Twitter to get the latest news!
  • Report an Issue: Report a new Issue

Develop and Contribute

If you want to get more involved by contributing to Milkman, join us here:

  • GitHub: Development takes place here!
  • Gitter: Chat with other project developers
  • Issue list: Discuss development issues around the project

You can find out how to contribute to Milkman in our Contribution Guidelines.
